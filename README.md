# Simulador de Rede Social

Projeto para o curso de estrutura de dados.

Utilizando grafos, algoritmos de busca e de ordenação, simula-se uma rede social onde usuarios podem seguir e serem seguidos por outros usuários da rede.

Além da estrutura geral, os seguintes objetivos foram implementados:

 - calcular o número de seguidores de um dado usuário;
 - calcular o número de usuários que um determinado usuário segue;
 - encontrar um caminho entre dois usuários;
 - dado o parâmetro inteiro k, ranquear os top k "influencers" da rede;
 - ordenar a lista de "Stories" de um dado usuário por ordem alfabética, com os "melhores amigos"  à frente dos demais amigos.
